Leo-API-Center

By Malay Bhavsar

Welcome to Leo-API-Center. Here I have a collection of APIs combined to one project which helps me in reducing the duplicate code and allows easy maintenance and improve productivity. Ihave converted the reusable codes to several functions which helps me to develop new API's much faster with less effort.

My Primary Goal is to develop APIs and optimize them for faster and better results. This APIs are made using ExpressJS framework. Current Number of APIs: 31

Path Type Require Token Body Description
Leo-Auth
/Auth/new_account POST No fname, lname, username, password, email, mobile, country Creates a New Account
/Auth/account GET Yes - Fetches details about that account
/Auth/ch_account POST Yes * Pass things to be changed E.g. fname, mobile etc but not username or password
/Auth/rm_account POST Yes username, password Delete your Account
/Auth/ch_password POST Yes old_password, new_password Update your Account Password
/Auth/login POST No username, password Login to your account and obtain Token
